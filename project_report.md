# 프로젝트 구조 및 내용 분석 보고서

## 1. 폴더 및 파일 구조

- 최상위:  
  - 다양한 Python 예제 및 실습 파일(`*.py`), 마크다운 문서(`README.md`, `0908_report.md`)
  - 주요 폴더:  
    - `function/` : 함수, 클래스, 예외처리 등 함수형/객체지향 예제
      - `Class/` : 클래스 관련 예제 다수
      - `Try/` : 예외처리 예제
    - `tuple_set_dic/` : 튜플, 세트, 딕셔너리 등 자료구조 예제

## 2. 주요 파일 및 예제 설명

- **README.md, 0908_report.md**  
  - 프로젝트 및 파이썬 기초 개념(변수, 연산자, 조건문 등) 정리

- **기본 문법 예제**  
  - `list_1.py`, `tuple_set_dic/dict_01.py` 등: 리스트, 딕셔너리 등 자료구조 사용법 실습
  - `bool_and_or.py`, `if_test.py` 등: 논리연산, 조건문 실습

- **함수/클래스/예외처리**  
  - `function/example_01.py` : 사용자 입력 처리 함수 예제
  - `function/Class/class_01.py` : 클래스 변수와 인스턴스 변수의 차이 실습
  - `function/Class/class_08.py` : 학생 클래스(이름, 국영수, 총점/평균/학점 메서드)
  - `function/Class/class_09.py` : 가위바위보 게임 클래스 구현
  - `function/Try/` : 예외처리 관련 예제

- **자료구조 심화**  
  - `tuple_set_dic/` 내 다양한 파일: 딕셔너리, 세트, 튜플의 생성, 접근, 활용법 실습

## 3. 전반적 특징 및 활용 목적

- 파이썬 기초 문법부터 객체지향, 예외처리, 자료구조까지 폭넓게 실습
